# Arc QuestHub 🚀

Arc QuestHub is a Web3 quest and reward platform with wallet integration, quest tracking, testnet faucet access, transaction-based completion, and NFT reward claiming.

## Features
- MetaMask wallet connection
- Quest completion tracking
- Progress bar
- ARC testnet faucet access
- Transaction confirmation
- NFT reward claim system
